# UI Generator: O Arquiteto Visual (Estilo Finnie)

Este skill é o guardião da estética do projeto Stela. Sua missão é garantir que o SaaS não pareça um sistema comum, mas sim uma ferramenta financeira de alto nível.

## 🎨 Princípios de Design

### 1. Estética "Clean & Modern"
- **Espaçamento (White Space)**: Seja generoso. Use `p-6` ou `p-8` em cards.
- **Arredondamento**: Use `rounded-2xl` (16px) como padrão para cards e botões grandes.
- **Sombras**: Use sombras sutis (`shadow-sm` evoluindo para `shadow-md` no hover) para dar profundidade sem poluir.

### 2. Cores e Contraste
- **Fundo**: `#F8F9FA`.
- **Cards**: `#FFFFFF`.
- **Destaque Financeiro**:
    - Verde vibrante para saldos positivos e entradas.
    - Vermelho sofisticado para despesas e alertas.
    - Roxo/Rosa para gradientes de KPI de crescimento.

### 3. Visualização de Dados
- **Gráficos**: Devem ser limpos, sem bordas desnecessárias ou eixos poluídos.
- **Tabelas**: Use linhas zebradas sutis ou apenas bordas inferiores finas. O foco deve ser no dado.

---

## 🛠️ Stack de UI
- **Framework**: Tailwind CSS.
- **Base**: Shadcn/UI (Customizado).
- **Ícones**: Lucide React.
- **Animações**: Framer Motion (para transições suaves de entrada e hover).

---

## 📋 Checklist de Qualidade UI
- [ ] O componente segue a paleta de cores definida?
- [ ] O arredondamento de borda é de 16px?
- [ ] Há espaçamento suficiente entre os elementos?
- [ ] A hierarquia tipográfica destaca os números financeiros?
- [ ] O design é responsivo (Mobile-first)?
